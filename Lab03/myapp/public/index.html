Answers for the lab 3 exercises:

  • 3.1
    • Node.js is the platform to create a javascript webserver to allow express to run inside it.
    • Meanwhile express is the framework/library which work on Node.js to build the server.
    • The purpose of packages.json is to store the metadata for a project and handle all dependencies in the project.
  • 3.2
    • These files are static because when we tried to update the content after publishing it to our webpage, the updated content is not shown. Basically only the first saved content will be shown on the web and would be unchanged (therefore static).
    • The server continues to serve up the hello world message because the root file of "app.js" sends the "Hello World" by default, before the get request function.
